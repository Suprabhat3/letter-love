# Concept
Users input small details (memories, traits, nicknames), and AI weaves them into a perfect, tear-jerking (or funny) love letter or poem.

## Experience
Elegant, editorial typography, slow-reveal animations, and "shareable" distinct URL cards.

## AI Integration
Text generation (LLM) to craft the content.

## Theme 
Pink color scheme, elegant typography, good animation, and editorial style.

## Tech Stack
- Next.js
- Tailwind CSS
- Shadcn UI
- OpenAI API
- Supabase
